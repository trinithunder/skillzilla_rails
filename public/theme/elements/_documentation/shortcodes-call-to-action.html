Call to action

Live example

Encourage your visitor to perform specific actions with attractive elements.
Concerning hover button effects, please see buttons documentation.

Basic markup

<section class="section-call-to-action">
  <div class="container">
    <div class="action-wrapper">
      <div class="action-text">
        <h3 class="action-title">Your action title</h3>
        <p class="action-subtitle">Your action subtitle (optional)</p>
      </div>
      <div class="action-btn">
        <a href="#" class="btn btn-lg btn-dark">Contact us</a>
      </div>
    </div>
  </div>
</section>

4 call to action sizes

Small size: add action-sm class to section-call-to-action elements.

Default size: no need to add class to section-call-to-action elements.

Small size: add action-lg class to section-call-to-action elements.

Small size: add action-hg class to section-call-to-action elements.

Bordered style

For bordered layout, add action-boxed action-bordered class to section-call-to-action element.

<section class="section-call-to-action action-boxed action-bordered">
  <div class="container">
    <div class="action-wrapper">
      <div class="action-text">
        <h3 class="action-title">Your action title</h3>
        <p class="action-subtitle">Your action subtitle (optional)</p>
      </div>
      <div class="action-btn">
        <a href="#" class="btn btn-lg btn-dark">Contact us</a>
      </div>
    </div>
  </div>
</section>

Centered layout

For centered layout, add action-center class to section-call-to-action element.

<section class="section-call-to-action action-center">
  <div class="container">
    <div class="action-wrapper">
      <div class="action-text">
        <h3 class="action-title">Your action title</h3>
        <p class="action-subtitle">Your action subtitle (optional)</p>
      </div>
      <div class="action-btn">
        <a href="#" class="btn btn-lg btn-dark">Contact us</a>
      </div>
    </div>
  </div>
</section>

This layout is compatible with previous style, for example action-center action-boxed action-bordered.
