Tables

Live example

This template use the default bootstrap table markup. You can read more about it here: http://getbootstrap.com/css/#tables.

Striped Table

Add table-striped class to your table to use this style.

<table class="table table-striped">
  <!-- TABLE CONTENT -->
</table>

Hover Table

If you want to add a hover effect, add table-hover class to your table to use this style.

<table class="table table-hover">
  <!-- TABLE CONTENT -->
</table>

Bordered Table

Add table-striped class to your table to use this style.

<table class="table table-bordered">
  <!-- TABLE CONTENT -->
</table>
