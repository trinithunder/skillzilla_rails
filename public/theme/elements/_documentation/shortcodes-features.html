Features boxes

Show up your services or products with beautiful boxes.

Live example

Basic markup

Default feature have a left icon, a title and a description:

<div class="feature-box">
  <div class="icon">
    <i class="nc-icon-outline ui-1_edit-72"></i>
  </div>
  <div class="feature-content">
    <h3 class="feature-title">Feature title</h3>
    <p>Feature description.</p>
  </div>
</div>

Icon can be a font like here, but it can be an image too.

Custom background icon color

Theme have 2 default background color, more gray and dark. But you can choose all colors of theme.

Gray icon: gray icon have hover effect. It becomes dark on hover

<div class="feature-box">
  <div class="icon icon-gray">
    <i class="nc-icon-outline ui-1_edit-72"></i>
  </div>
  <div class="feature-content">
    <h3 class="feature-title">Feature title</h3>
    <p>Feature description.</p>
  </div>
</div>

Dark icon:

<div class="feature-box">
  <div class="icon icon-dark">
    <i class="nc-icon-outline ui-1_edit-72"></i>
  </div>

Square layout

For square style, add icon-square to icon element:

<div class="feature-box">
  <div class="icon icon-square">
    <i class="nc-icon-outline ui-1_edit-72"></i>
  </div>
  <div class="feature-content">
    <h3 class="feature-title">Feature title</h3>
    <p>Feature description.</p>
  </div>
</div>

Icon with border line

For border style, add icon-line to icon element:

<div class="feature-box">
  <div class="icon icon-line">
    <i class="nc-icon-outline ui-1_edit-72"></i>
  </div>
  <div class="feature-content">
    <h3 class="feature-title">Feature title</h3>
    <p>Feature description.</p>
  </div>
</div>

This is compatible with square icon: icon icon-square icon-line class.

Icon with no background

If you don't want background, add icon-simple to icon element:

<div class="feature-box">
  <div class="icon icon-simple">
    <i class="nc-icon-outline ui-1_edit-72"></i>
  </div>
  <div class="feature-content">
    <h3 class="feature-title">Feature title</h3>
    <p>Feature description.</p>
  </div>
</div>

Right align icon

For right icon, add feature-icon-right to feature-box element and place your icon after your feature content:

<div class="feature-box feature-icon-right">
  <div class="feature-content">
    <h3 class="feature-title">Feature title</h3>
    <p>Feature description.</p>
  </div>
  <div class="icon">
    <i class="nc-icon-outline ui-1_edit-72"></i>
  </div>
</div>

Small size

For small feature size, add feature-sm to feature-box element:

<div class="feature-box feature-sm">
  <div class="icon icon-simple">
    <img src="assets/img/icons/business-64px-color_chart.png" alt="chart">
  </div>
  <div class="feature-content">
    <h3 class="feature-title">Feature title</h3>
    <p>Feature description.</p>
  </div>
</div>

As you can see here, icon can be an image too.

Center align

To center your feature, add center to feature-box element:

<div class="feature-box center">
  <div class="icon icon-line">
    <i class="nc-icon-glyph business_hierarchy-53"></i>
  </div>
  <div class="feature-content">
    <h3 class="feature-title">Feature title</h3>
    <p>Feature description.</p>
  </div>
</div>

You can add a border to your center align feature. Just add center to feature-box element:

<div class="feature-box boxed center">
  <div class="icon icon-line">
    <i class="nc-icon-glyph business_hierarchy-53"></i>
  </div>
  <div class="feature-content">
    <h3 class="feature-title">Feature title</h3>
    <p>Feature description.</p>
  </div>
</div>
