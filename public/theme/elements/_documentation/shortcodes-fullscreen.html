Fullscreen page

Create Beautiful Fullscreen Scrolling Websites.

Live example

Theme use Fullpage.js plugin. For more information about this plugin, please visit Fullpage official website

Basic markup

To create full page layout, add fullpage class to main-content. Then wrap your different page inside section:

<div id="main-content" class="fullpage">
  <section class="section" data-title="First page">
    <!-- SECTION 1 CONTENT -->
  </section>
  <section class="section" data-title="Second page">
    <!-- SECTION 2 CONTENT -->
  </section>
  <section class="section" data-title="Third page">
    <!-- SECTION 3 CONTENT -->
  </section>
</div>

Navigation title for each page is managed via data-title attribute.

If you add section-dark class to your section, navigation color will automatically change.
