Boxed & wide layout

Theme is wide by default. You can choose a boxed layout too. In this case, your page will be "boxed" for screen width superior to 1240px.You can add a custom background like a big image, a pattern image or just a background color.

Live example

Create boxed layout

To have a boxed layout, just add boxed class to your body.

<body class="boxed">

Image background

To add a large background image, add data-bg-img attribute to body with your image path.

<body class="boxed" data-bg-img="your-background-image.jpg">

Pattern background

For pattern image, add bg-pattern class to your body in order to repeat pattern (which is not the case of big background image.

<body class="boxed bg-pattern" data-bg-img="your-background-pattern.jpg">
