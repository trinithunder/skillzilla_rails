Post Comments

Post can have html comments, or load comments via Facebook or Disqus.

Facebook comments

Live example

It's very simple to add Facebook comments since it's just 1 line of code.

<section class="post-comments" id="comments">
  <div class="container">
    <div class="row">
      <div class="col-sm-8 col-sm-offset-2">
        <div class="fb-comments" data-href="current-url.html" data-width="100%" data-numposts="5"></div>
      </div>
    </div>
  </div>
</section>

Facebook comments have various settings, such as number of comments display with data-numposts.
You will find additional information of Facebook plugin documentation here.

NB: Don't forget to include Facebook script as well at the bottom of your page. To get it, click on get code button in Facebook comments plugin page.

Facebook script looks like this (don't use it, it's just an example):

<div id="fb-root"></div>
<script>(function(d, s, id) {
  var js, fjs = d.getElementsByTagName(s)[0];
  if (d.getElementById(id)) return;
  js = d.createElement(s); js.id = id;
  js.src = "//connect.facebook.net/fr_FR/sdk.js#xfbml=1&version=v2.4&appId=699253173491854";
  fjs.parentNode.insertBefore(js, fjs);
}(document, 'script', 'facebook-jssdk'));</script>

Disqus comments

Live example

In the same way as Facebook comments, Disqus comments are easy to use. Include 1 line of code and specific script

<section class="post-comments" id="comments">
  <div class="container">
    <div class="row">
      <div class="col-sm-8 col-sm-offset-2">
        <div id="disqus_thread"></div>
      </div>
    </div>
  </div>
</section>

Facebook comments have various settings, such as number of comments display with data-numposts.
You will find additional information of Facebook plugin documentation here.

NB: Don't forget to include Disqus script as well at the bottom of your page. To get it, create an account on Disqus website here.

Disqus script looks like this (don't use it, it's just an example):

<script type="text/javascript">
/* CONFIGURATION VARIABLES, REPLACE WITH YOUR REAL DISQUS SHORTNAME */
var disqus_shortname = 'your-disqus-name';
/* DON'T EDIT BELOW THIS LINE */
(function() {
    var dsq = document.createElement('script'); dsq.type = 'text/javascript'; dsq.async = true;
    dsq.src = '//' + disqus_shortname + '.disqus.com/embed.js';
    (document.getElementsByTagName('head')[0] || document.getElementsByTagName('body')[0]).appendChild(dsq);
})();
</script>
