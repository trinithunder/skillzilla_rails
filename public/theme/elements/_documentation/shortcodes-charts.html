Charts

Simple, clean and engaging charts.

Live example

Theme use ChartJS plugin. For more information about this plugin, please visit ChartJS official website

Html markup

Html markup is very simple. Just add a canvas element with a specific id like that:

<canvas id="my-chart"></canvas>

Create your chart with javascript

To create a chart, we need to instantiate the Chart id. You will find all information in official plugin website here. You can check js file that manage charts shortcode page too assets/js/charts.js file. Your code will look like this:

var donutData = [
    {value: 300, color:"rgba(226, 120, 93,0.9)", highlight: "rgba(226, 120, 93,1)", label: "Orange"},
    {value: 40, color: "rgba(54, 173, 199,0.9)", highlight: "rgba(54, 173, 199,1)", label: "Blue"},
    {value: 100,color: "rgba(255, 200, 112,0.9)", highlight: "rgba(255, 200, 112,1)", label: "Yellow"}
];  
window.myPie = new Chart(document.getElementById("donut-chart").getContext("2d")).Pie(donutData, {
    responsive: true,
    tooltipCornerRadius: 0
});
