Live example

Theme includes various font: Open Sans (default), Satisfy (handwritting), Sumana (typing style) and Montserrat for headings.

How to use custom fonts?

To use Sumana font, add font-sumana class to your element.

<div class="font-sumana">
  <p>Your text here.</p>
  <p>Other text here.</p>
</div>

To use Satisfy font, use font-handwriting class.

To use Satisfy font, use font-handwriting class.

To use Montserrat font, use font-montserrat class.

Add new fonts

You can add new fonts. Easiest way is to use Google fonts: https://www.google.com/fonts.

For example, to add Roboto font, add this code in head of theme:

<link href='https://fonts.googleapis.com/css?family=Roboto' rel='stylesheet' type='text/css'>

Then, you can create a new class in your css file, for example:

.font-roboto {font-family: 'Roboto',arial;}

Now you just have to wrap your text inside this class, like previously:

<div class="font-roboto">
  <p>Your text here.</p>
  <p>Other text here.</p>
</div>
