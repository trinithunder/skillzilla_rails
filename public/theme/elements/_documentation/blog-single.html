Single Post Layouts

Post have multiple headers variations: image, slider, video, audio, quote or simple title.
Post can have left or right sidebar, no sidebar, or 2 sidebars on left.
Post can have html comments, or load comments via Facebook or Disqus (see comments section).

Global structure

All post have same global structure with header, post content, post navigation and comments.

<div id="main-content" class="post-single">
  <!-- BEGIN HEADER -->
  <header class="section-header">
    <!-- HEADER CONTENT -->
  </header>
  <!-- END HEADER -->
  
  <!-- BEGIN POST -->
  <section class="section">
    <!-- POST CONTENT -->
  </section>
  <!-- END POST -->

  <!-- BEGIN POST NAV -->
  <section class="post-nav">
    <div class="container">
      <div class="row">
        <div class="col-xs-6">
          <a class="previous" href="previous-post.html">
            <span class="prev-icon">
              <i class="nc-icon-glyph arrows-1_bold-left"></i>
            </span>
            <span>Previous Post</span>
            <span class="post-nav-title">Previous post title</span>
          </a>
        </div>
        <div class="col-xs-6">
          <a class="next" href="next-post.html">
            <span>Next Post</span>
            <span class="next-icon">
              <i class="nc-icon-glyph arrows-1_bold-right"></i>
            </span>
            <span class="post-nav-title">Next post title</span>
          </a>
        </div>
      </div>
    </div>
  </section>
  <!-- END POST NAV -->

  <!-- BEGIN POST COMMENTS -->
  <section class="post-comments">
    <!-- COMMENTS CONTENT -->
  </section>
  <!-- END POST COMMENTS -->

</div>

Simple header with title

Live example

If you just want a header with title and some metadatas (author, date of post, category):

<div id="main-content" class="post-single">
  <header class="section-header">
    <div class="container top-element">
      <div class="row top-text">
        <h1 class="blog-title">Post Title</h1>

            by <a href="#">Admin</a>
          </li>
          <li>
            on <a href="#">January 15, 2016</a>
          </li>
          <li>
            in <a href="#">Blog Category</a>
          </li>
        </ul>
      </div>
    </div>
  </header>
  <!-- POST CONTENT, POST NAV, COMMENTS -->
</div>

You want a parallax effect on page scroll? Theme use Skrollr plugin. Here is an example to add this cool effect:

<header class="section-header header-lg header-dark img-cover" data-bg-img="your-image-path.jpg" data-0="background-position:0px 0px;" data-500="background-position:0px -100px;">
  <!-- HEADER CONTENT -->
</header>

Header with html video

Live example

You can add html video in your header.

<header class="section-video header-dark">
  <div class="audio-video-wrapper">
    <video id="video-js" autoplay class="video-js full-video vjs-sublime-skin" controls preload="auto" height="400" poster="path-poster-img.jpg">
      <source src="path-video.mp4" type="video/mp4">
    </video>
  </div>
  <div class="container top-element">
    <div class="row top-text">
      <h1 class="blog-title">Html Video Post</h1>
      <ul class="blog-meta">
        <li>
          by <a href="#">Admin</a>
        </li>
        <li>
          on <a href="#">January 15, 2016</a>
        </li>
        <li>
          in <a href="#">Blog Category</a>
        </li>
      </ul>
    </div>
  </div>
</header>

If you don't want that video play automatically when page is loaded, just remove autoplay attribute in video.
You can also add html video, or embed video (Youtube & Vimeo) inside your post content.
