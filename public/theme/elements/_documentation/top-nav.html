Top Navigation

Top navigation have a large set of options and possibilities.

Top navigation can have a transparent, light or dark background, with light or dark text color, submenu light or dark.

You can add a topbar too with various widgets.

Live example

Basic markup

Here is the basic markup of top navigation with menu align right.

<header id="main-navigation">
  <div class="main-nav-wrapper">
    <div class="container-fluid">
      <div class="nav-left">
        <div id="logo">
          <!-- YOUR LOGO IMAGE -->
        </div>
      </div>
      <div class="nav-right">
        <nav id="main-menu">
          <!-- YOUR MENU -->
        </nav>
        <ul class="nav-tools">
        </ul>
      </div>
    </div>
  </div>
</header>

Customize navigation background color

To have transparent navigation, add header-transparent class to your body:

<body class="header-transparent">

To have navigation with light background, add header-light class to your body:

<body class="header-light">

To have transparent navigation, add header-dark class to your body:

<body class="header-dark">

By default, when you scroll page, background is white.
To have dark background when you scroll page, add header-scroll-dark class to your body:

<body class="header-scroll-dark">

Customize background submenu color

By default, submenu background is white.
To have submenu with dark background, add submenu-dark class to your main-navigation id:

<header id="main-navigation" class="submenu-dark">

Basic submenu markup

Basic submenu display as a vertical list. You can add as many levels as you want.

<ul>
  <li>
    <a href="#">Menu title</a>
    <ul>
      <li class="submenu">
        <a href="link-1">Submenu title</a>
        <ul>
          <li class="submenu">
            <a href="link-2">Sub submenu title</a>
            <ul>
              <li>
                <a href="link-3">Sub submenu 1</a>
              </li>
            </ul>
          </li>
        </ul>
      </li>
    </ul>
  </li>
</ul>

Megamenu submenu markup

Megamenu content will display as an horizontal list. You can add up to 5 horizontal columns per menu.

<ul>
  <li class="mega-menu">
    <a href="#">Menu</a>
    <div class="mega-menu-content col-5 clearfix">
