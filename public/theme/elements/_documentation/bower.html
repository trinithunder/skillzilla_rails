Bower Documentation

How to install Bower?

Bower requires node, npm and git. Bower is a command line utility. Install it with npm: npm install -g bower

If you want to change folder in which packages are installed, you can customize .bowerrc file. You will find this file in bower folder.

Commands

bower init

This will intialize bower.

bower install

This will install all packages listed in bower.json file.

bower search bootstrap

This will search all packages related to Bootstrap. Here Bootstrap is an example. You can change it with what you want: jquery, masonry, select2...

bower install bootstrap -save

This will install bootstrap package and add the package to bower.json file.

bower uninstall bootstrap -save

This will remove bootstrap package and add the package to bower.json file.

Troubleshooting

If you have a problem when installing dependencies or running Bower commands:

  1. Delete the js/libs directory generated by bower
  2. Rerun bower install command
