Ajax project loading

Instead of having classic navigation with new projects list on different pages, you can display new projects in the same page using ajax.
You have 2 possibilities 2 load more projects: via a link (with a click event) or directly when user scroll to bottom of the page.
Moreover, you can load project content inside page.

Ajax project in page

Live example

You can load a specific project in a dedicated page, or directly inside projects list page using Ajax.
Here is an example of projects list with 3 columns and caption visible:

<section class="section-portfolio">
  <div class="container">
    <div class="masonry grid grid-3 with-caption portfolio-ajax">
      <div class="item" data-content="project-1-path.html">
        <div class="item-wrapper">
          <figure class="indigo darken-3 he-2 caption-visible caption-center">
            <img src="project-1-img.jpg" alt="">
            <div class="hover-icons">
              <div class="hover-icons-wrapper">
                <a href="#" class="ajax-link">
                  <i class="nc-icon-outline media-1_polaroid-multiple"></i>
                </a>
              </div>
            </div>
            <figcaption>
              <h4><span>Project 1 title</span></h4>
              <p>tag 1, tag 2</p>
            </figcaption>
          </figure>
        </div>
      </div>
      <div class="item" data-content="http://localhost/new/include/ajax-portfolio/portfolio-ajax-image.php">
        <!-- ITEM 2 CONTENT -->
      </div>
    </div>
  </div>
</section>

You can load various project type: project with simple image, with images sliders, with video...

Ajax projects on click

Live example

You have a button at the button of the page and new projects will load when user click on it. Your ajax page need to have same layout as you current page of course. Here is the markup with 3 projects columns:

<section class="section-portfolio">
  <div class="container">
    <div class="masonry grid grid-3 infinite-container infinite-onclick">
      <div class="item">
        <!-- ITEM 1 CONTENT -->
      </div>
      <div class="item">
        <!-- ITEM 2 CONTENT -->
      </div>
      <div class="item">
        <!-- ITEM 3 CONTENT -->
      </div>
    </div>
    <div class="next-page-wrapper">
      <a data-num-page="3" href="include/ajax-portfolio/portfolio-infinite-scroll" class="btn btn-dark btn-rounded btn-bordered btn-lg next-page">Load More...</a>
    </div> 
  </div>
</section>

The main difference between classic layout with html pagination is specific classes for projects:infinite-container and infinite-onclick. And of course, the link at the bottom of the page to load more post with next-page class.

Ajax projects on scroll

Live example

Instead of a text link at the button of the page, we place a loader link in order that user can see that new projects are loading and will appear.
Here new projects will load automatically when you reach bottom of the page. Here is the markup with 3 projects columns:

<section class="section-portfolio">
  <div class="container">
    <div class="masonry grid grid-3 infinite-container">
      <div class="item">
        <!-- ITEM 1 CONTENT -->
      </div>
      <div class="item">
        <!-- ITEM 2 CONTENT -->
      </div>
      <div class="item">
        <!-- ITEM 3 CONTENT -->
      </div>
    </div>
    <div class="next-page-wrapper">
      <a href="include/ajax-portfolio/portfolio-infinite-scroll" class="next-page" data-num-page="3"></a>
      <div class="spinner">
        <div class="double-bounce1"></div>
        <div class="double-bounce2"></div>
      </div>
    </div> 
  </div>
</section>
