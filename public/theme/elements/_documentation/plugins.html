Plugins list

Themes use a large set of plugins to fit all needs. But you will may not need all functionnalities and may want to remove some of plugins unnecessary. By default all files are minified to speed up page loading. So will only have a min.css and a min.js file in your page. To see list of all css and js files of the theme, you can open page-blank.html file with your favorite editor.

Which files to include in my custom page?

We make various css files in order to allow you not inlude unnecessary code. So for example, for a team page, you will add team.css file. For a restaurant page, you will add restaurant.css file.


Here is a list of files you have to include depending of what you need in your page.

Revolution slider

Css files in your head:

<link rel="stylesheet" href="assets/js/plugins/revolution-slider/revolution/css/settings.css"/>
<link rel="stylesheet" href="assets/js/plugins/revolution-slider/revolution/css/navigation.css"/>
<link rel="stylesheet" href="assets/css/sliders.css"/>

Js files at the end of your page:

<script src="assets/js/plugins/revolution-slider/revolution/js/jquery.themepunch.tools.min.js?rev=5.0"></script>
<script src="assets/js/plugins/revolution-slider/revolution/js/jquery.themepunch.revolution.min.js?rev=5.0"></script>
<script src="assets/js/sliders.js"></script>
