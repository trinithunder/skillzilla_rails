Blockquotes

Carefully craft typography to create beautiful design

Live example

Basic markup

By default, a blockquote have a sentence / paragraph, an author and a quote sign.

<blockquote>
  <p>Two things are infinite: the universe and human stupidity; and I'm not sure about the universe.</p>
  <div class="quote-author">Albert Einstein</div>
</blockquote>

Icon can be a font like here, but it can be an image too.

Background color

You can add all background color you want. First add blockquote-bg class to blockquote in order to add padding to it. Then, add your background color with theme colors or using data-bg-color attribute.

<blockquote class="t-light blockquote-bg" data-bg-color="#3D3D3D">
  <p>The flower that blooms in adversity is the rarest and most beautiful of all.</p>
  <div class="quote-author">Walt Disney Company, Mulan</div>
</blockquote>

Big size

To create big blockquote, add blockquote-lg class to your blockquote.

<blockquote class="blockquote-lg">
  <p>Success is getting what you want, happiness is wanting what you get.</p>
  <div class="quote-author">W.P. Kinsella</div>
</blockquote>

Right align

For right align blockquote, add blockquote-right class to your blockquote.

<blockquote class="blockquote-right">
  <p>The only true wisdom is in knowing you know nothing.</p>
  <div class="quote-author">Socrates</div>
</blockquote>
