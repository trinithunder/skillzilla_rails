Getting Started

Here you will find all informations to quickly start your project. You will have a global idea about files organization.

What's included

Once you have downloaded the theme and unzipped it, you will find the following folders structure:

  • _documentation
  • assets
    • css
      • min
        • css.min.css
    • fonts
    • img
    • js
      • libs
      • min
      • plugins
    • videos
  • include
  • save-your-time

Folder: assets

Here you will find all files necessary in order to make theme works: css, images, javascript and all plugins... You will find minified version of css and js files too to speed up page loading.

Folder: include

This folder contains all ajax content, like blog post, portfolio project or ecommerce product quickview.

Folder: pages

This folder contains all the demo pages files that you can see in the live version, you may wish to import code to your project from that. This is only used for reference and we recommend to start your project from page-blank.html file.

Folder: save-time

Theme support 3 popular build systems called Bower, Grunt and Gulp. Grunt & Gulp will allow you to automate repetitive tasks like minification, compilation or testing. Moreover, Bower will allow to download latest version of all plugins in seconds.
