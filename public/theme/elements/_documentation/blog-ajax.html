Ajax posts loading

Instead of having classic navigation with posts on different pages, you can display new posts in the same page using ajax.

You have 2 possibilities 2 load more posts: via a link (with a click event) or directly when user scroll to bottom of the page.

Ajax posts on click

Live example

You have a button at the button of the page and new posts will load when user click on it. Your ajax page need to have same layout as you current page of course. Here is the markup with 3 posts columns:

<section class="section-blog">
  <div class="container-fluid">
    <div class="posts masonry masonry-layout grid grid-3 grid infinite-container infinite-onclick">
      <div class="item">
        <div class="item-wrapper">
          <div class="post">
            <div class="post-medias">
              <!-- MEDIA CONTENT: IMAGES, AUDIO, VIDEO -->
            </div>
            <div class="post-info">
              <h2 class="post-title"><a href="blog-single-left.html">Post title</a></h2>
              <ul class="post-meta">
                <li>
                  by <a href="#">Admin</a>
                </li>
                <li>
                  on <a href="#">January 15, 2016</a>
                </li>
              </ul>
              <div class="post-excerpt">
                <p>Post excerpt.</p>
              </div>
              <a href="" class="more">Read More</a>
            </div>
          </div>
        </div>
      </div>
      <div class="item">
        <!-- ITEM 2 -->
      </div>
    </div> 
    <div class="next-page-wrapper">
      <a href="include/ajax-blog/blog-infinite-scroll" class="btn btn-lg btn-rounded btn-bordered btn-dark next-page" data-num-page="3">Load More</a>
    </div>  
  </div>
</section>

The main difference between classic layout with html pagination is specific classes for posts:infinite-container and infinite-onclick. And of course, the link at the bottom of the page to load more post with next-page class.

Ajax posts on scroll

Live example

Instead of a text link at the button of the page, we place a loader link in order that user can see that new posts are loading and will appear.
Here new posts will load automatically when you reach bottom of the page. Here is the markup with 3 posts columns:

<section class="section-blog">
  <div class="container-fluid">
    <div class="posts masonry masonry-layout grid grid-3 grid infinite-container">
      <div class="item">
        <div class="item-wrapper">
          <div class="post">
            <div class="post-medias">
              <!-- MEDIA CONTENT: IMAGES, AUDIO, VIDEO -->
            </div>
            <div class="post-info">
              <h2 class="post-title"><a href="blog-single-left.html">Post title</a></h2>
              <ul class="post-meta">
                <li>
                  by <a href="#">Admin</a>
                </li>
                <li>
                  on <a href="#">January 15, 2016</a>
                </li>
              </ul>
              <div class="post-excerpt">
                <p>Post excerpt.</p>
              </div>
              <a href="" class="more">Read More</a>
            </div>
          </div>
        </div>
      </div>
      <div class="item">
        <!-- ITEM 2 -->
      </div>
    </div> 
    <div class="next-page-wrapper">
      <a href="include/ajax-blog/blog-infinite-scroll" class="next-page" data-num-page="3"></a>
      <div class="spinner">
        <div class="double-bounce1"></div>
        <div class="double-bounce2"></div>
      </div>
    </div> 
  </div>
</section>
