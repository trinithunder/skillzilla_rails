Buttons

Live example

Buttons come with various sizes, styles, backgrounds and effects. You can use button, link or other element to create button.

4 Buttons sizes

Add a class like btn-sm to your button to change size:

Small size: 
<button type="button" class="btn btn-sm">See More</button>
Default size: 
<button type="button" class="btn">See More</button>
Large size: 
<button type="button" class="btn btn-lg">See More</button>
Huge size: 
<button type="button" class="btn btn-hg">See More</button>

3 Buttons styles

Add a class like btn-square to your button to change style:

Default style (2px border-radius): 
<button type="button" class="btn">See More</button>
Square style: 
<button type="button" class="btn btn-square">See More</button>
Rounded style: 
<button type="button" class="btn btn-rounded">See More</button>

Embossed variation

For each previous style, you can add an embossed effect, for example:

<button type="button" class="btn btn-embossed">See More</button>

Bordered variation

For each previous style, you can add an bordered effect (not compatible with embossed effect), for example:

<button type="button" class="btn btn-bordered">See More</button>

3 Hover effects

You can add effects with icons. Icon can appear to the left, right or from top side.

Left icon effect 
<button class="btn btn-lg icon-left-effect"><i class="nc-icon-outline media-1_camera-18"></i><span>Take Picture</span></button>
Right icon effect 
<button class="btn btn-lg icon-right-effect"><i class="nc-icon-outline media-1_camera-18"></i><span>Take Picture</span></button>
Top icon effect 
<button class="btn icon-top-effect btn-icon-settings"><span>Settings</span></button>
