Blog

Blog have multiple layouts: modern, newspaper, timeline, or classic with large and small images.

For figure markup, please refer to hover effects section here

Timeline layout

Timeline have 3 layouts: timeline alternate, left sidebar and right sidebar.

Modern layout

Live example

Here is the basic markup of modern blog layout:

<section class="section-blog">
  <div class="posts masonry grid grid-4 no-space">
    <div class="item animated" data-animation="zoomIn" data-animation-delay="400">
      <div class="item-wrapper">
        <!-- FIGURE CONTENT -->
      </div>
    </div>
    <div class="item item-wide item-tall animated" data-animation="zoomIn" data-animation-delay="600">
      <div class="item-wrapper">
        <!-- FIGURE CONTENT -->
      </div>
    </div>
  </div>
</section>

If you want to have a black & white effect on mouseover for images, add black-white-effect class to your figures.

Newspaper layout

Live example

Here is the basic markup of newspaper blog layout with 3 columns:

<section class="section-blog">
<div id="blog-main" class="container-large">
  <div class="blog-wrapper">
    <ul class="items-filter">
      <!-- FILTERS LIST -->
    </ul>
    <div class="posts masonry masonry-layout grid grid-3">
      <div class="item">
        <div class="item-wrapper">
          <div class="post">
            <div class="post-medias">
              <!-- MEDIA CONTENT: IMAGES, AUDIO, VIDEO -->
            </div>
            <div class="post-info">
              <h2 class="post-title"><a href="blog-single-left.html">Post title</a></h2>
              <ul class="post-meta">
                <li>
                  by <a href="#">Admin</a>
                </li>
                <li>
                  on <a href="#">January 15, 2016</a>
                </li>
              </ul>
              <div class="post-excerpt">
                <p>Post excerpt</p>
              </div>
              <a href="" class="more">Read More</a>
            </div>
          </div>
        </div>
      </div>

Filtering items

Basic markup for filtering items:

<ul class="items-filter">
  <li>
    <a href="#" data-filter=".filter-1">Filter 1</a>
  </li>
  <li>
    <a href="#" data-filter=".filter-2">Filter 2</a>
  </li>
  <li>
    <a href="#" data-filter=".filter-3">Filter 3</a>
  </li>
  <li>
    <a href="#" data-filter=".filter-4">Filter 4</a>
  </li>
  <li class="filter-all">
    <a href="#" data-filter="*" class="current">All Categories</a>
  </li>
</ul>

For filter posts by category, add category class to each post. For example, if you have a filter "design" and want to make a post in this category, add design class to your item

<div class="item design">
  <!-- ITEM CONTENT -->
</div>

Pagination

Place your pagination html markup right after your posts list

<div class="posts">
  <!-- YOUR POSTS LIST -->
</div>
<nav class="pagination-wrapper">
  <ul class="pagination">
    <li class="active"><a href="#">1</a></li>
    <li><a href="#">2</a></li>
    <li><a href="#">3</a></li>
    <li><a href="#">4</a></li>
    <li><a href="#">5</a></li>
    <li>
      <a href="#" aria-label="Next">
        <span aria-hidden="true"><i class="nc-icon-outline arrows-1_tail-right"></i></span>
      </a>
    </li>
  </ul>
</nav>

For filter posts by category, add category class to each post. For example, if you have a filter "design" and want to make a post in this category, add design class to your item

<div class="item design">
  <!-- ITEM CONTENT -->
</div>
