See live example

  <div class="plan-wrapper">
    <div class="plan-header">

      <div class="price">
        <div class="price-wrapper">
          <span class="price-symbol">$</span>
          <span class="price-value">17</span>
        </div>
        <div class="price-description">
          / month
        </div>
      </div>
    </div>
    <div class="plan-info">
      <div class="plan-features-txt">
        Quick description
      </div>
      <div class="plan-features-list">
        <ul>
          <li><i class="nc-icon-outline ui-1_check"></i> Element 1</li>
          <li><i class="nc-icon-outline ui-1_check"></i> Element 2</li>

  </div>
</div>

For example, for 3 packages using this previous markup, you will have:

<div class="row">
  <div class="col-md-4">

  </div>
  <div class="col-md-4">
    <!-- PACKAGE 2 MARKUP -->           
  </div>
  <div class="col-md-4">
    <!-- PACKAGE 3 MARKUP -->             
  </div>
</div>

Dark layout

For dark layout, add plan-dark class to price-plan element.

<div class="price-plan plan-dark plan-boxed">
  <!-- PLAN CONTENT MARKUP -->
</div>

Collapsed variation

To use this design, add plans-collapsed class to parent element:

<div class="row plans-collapsed">
  <div class="col-md-4">
    <!-- PACKAGE 1 MARKUP -->        
  </div>
  <div class="col-md-4">
    <!-- PACKAGE 2 MARKUP -->            
  </div>
  <div class="col-md-4">
    <!-- PACKAGE 3 MARKUP -->           
  </div>
</div>
