Tabs

Live example

Tabs allow to display grouped information and show only necessary content.

Basic markup

<ul class="nav nav-tabs" role="tablist">
  <li role="presentation" class="active">
    <a href="#tab-1" aria-controls="tab-1" role="tab" data-toggle="tab">
      Tab 1
    </a>
  </li>
  <li role="presentation">
    <a href="#tab-2" aria-controls="tab-2" role="tab" data-toggle="tab">
      Tab 2
    </a>
  </li>
  <li role="presentation">
    <a href="#tab-3" aria-controls="tab-3" role="tab" data-toggle="tab">
      Tab 3
    </a>
  </li>
</ul>
<div class="tab-content">
  <div role="tabpanel" class="tab-pane fade in active" id="tab-1">
    Tab 1 content
  </div>
  <div role="tabpanel" class="tab-pane fade" id="tab-2">
    Tab 2content
  </div>
  <div role="tabpanel" class="tab-pane fade" id="tab-3">
    Tab 3 content
  </div>
</div>

Side layout

If you want to move tabs navigation to left side, wrap your tabs inside tab-left class element.

<div class="tab-left">
  <ul class="nav nav-tabs" role="tablist">
    <li role="presentation" class="active">
      <a href="#tab-1" aria-controls="tab-1" role="tab" data-toggle="tab">
        Tab 1
      </a>
    </li>
    <li role="presentation">
      <a href="#tab-2" aria-controls="tab-2" role="tab" data-toggle="tab">
        Tab 2
      </a>
    </li>
    <li role="presentation">
      <a href="#tab-3" aria-controls="tab-3" role="tab" data-toggle="tab">
        Tab 3
      </a>
    </li>
  </ul>
  <div class="tab-content">
    <div role="tabpanel" class="tab-pane fade in active" id="tab-1">
      Tab 1 content
    </div>
    <div role="tabpanel" class="tab-pane fade" id="tab-2">
      Tab 2content
    </div>
    <div role="tabpanel" class="tab-pane fade" id="tab-3">
      Tab 3 content
    </div>
  </div>
</div>

You can mix side layout with other style, for example buttons design

<div class="tab-left tab-buttons">
  <ul class="nav nav-tabs" role="tablist">
    <!-- TABS NAVIGATION -->
  </ul>
  <div class="tab-content">
    <!-- TABS CONTENT -->
  </div>
</div>
