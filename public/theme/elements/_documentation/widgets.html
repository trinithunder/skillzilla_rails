Facebook page widget

You can change your facebook page easily directly in your html page. Just change data-href attribute of your fb-page element:

<div class="fb-page" data-href="https://www.facebook.com/Themes-Lab-432272373573921" data-small-header="false" data-adapt-container-width="true" data-hide-cover="false" data-show-facepile="true" data-show-posts="false"></div>

Flickr widget

To modify Flickr stream, edit your widget inside your html page. You can customize user as well as number of images:

<div id="widget-flickr" class="widget widget-flickr" data-count="12" data-user="here-what-you-want">
    <h3 class="widget-title">Flickr Photos</h3>
    <ul class="flickr-list"></ul>
</div>

Dribbble widget

Dribbble widget use Jribbble plugin: https://github.com/tylergaw/jribbble.

To change Dribbble token, edit assets/js/widget.js file line 2.

Pinterest widget

Edit your widget directly in your html page by changing href attribute. For more information about all Pinterest widgets, please visit official website here: Pinterest widgets.
