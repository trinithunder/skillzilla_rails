Portfolio

Portfolio have multiple layouts and possibilities: masonry or grid, boxed or fullwidth, with space between item or not, with caption below image or not.. You can mix each one with another, which give you 16 variations. Each variation can have 2, 3, 4 or 5 columns. Number of columns decrease when screen size is lower.

Concerning hover effects possibilities, please refer to hover effects section here

Boxed / wide layout

For fullwidth style, add container-fluid class after section-portfolio element:

Live example

<section class="section-portfolio">
  <div class="container-fluid">
    <ul class="items-filter p-l-10">
      <!-- ITEMS FILTERS CONTENT -->
    </ul>
    <div class="masonry grid grid-4 with-caption">
      <!-- ITEMS LIST CONTENT -->
    </div>
  </div>
</section>

For boxed style, replace container-fluid by container class after section-portfolio element:

Live example

<section class="section-portfolio">
  <div class="container">
    <ul class="items-filter p-l-10">
      <!-- ITEMS FILTERS CONTENT -->
    </ul>
    <div class="masonry grid grid-4 with-caption">
      <!-- ITEMS LIST CONTENT -->
    </div>
  </div>
</section>

Item with space / collapsed

Portfolio item images can have padding or can be collapsed. For items with space (default layout), markup is the same as previous example:

<div class="masonry grid grid-4">
    <!-- ITEMS LIST CONTENT -->
  </div>
</div>

For collapsed style (no padding), just add no-space class to your masonry element:

<div class="masonry grid grid-4 no-space">
    <!-- ITEMS LIST CONTENT -->
  </div>
</div>

Portfolio filters

You can have filters on your portfolio page (or not)

<section class="section-portfolio">
  <div class="container-fluid">
    <ul class="items-filter p-l-10">
      <li>
        <a href="#" data-filter=".cat-1">Category 1</a>
      </li>
      <li>
        <a href="#" data-filter=".cat-2">Category 2</a>
      </li>
      <li>
        <a href="#" data-filter=".cat-3">Category 3</a>
      </li>
      <li class="filter-all p-r-10">
        <a href="#" data-filter="*" class="current">All Categories</a>
      </li>
    </ul>
    <div class="masonry grid grid-4">
      <!-- ITEMS LIST -->
    </div>  
  </div>
</section>

Then add specific filter class to each item. For example add cat-1 class for an item in category 1.

Masonry Layout

Live example

Here is the basic markup of masonry layout with 4 columns and title hover image:

<section class="section-portfolio">
  <div class="container-fluid">
    <ul class="items-filter p-l-10">
      <!-- ITEMS FILTERS CONTENT -->
    </ul>
    <div class="masonry grid grid-4">
      <div class="item">
        <div class="item-wrapper">
          <figure class="he-1 red darken-3" data-link="project-link-1.html">
            <img src="project-1-img.jpg" alt="">
            <figcaption>
              <h4>Project 1</h4>
              <p>tag 1, tag 2, tag 3</p>
            </figcaption>
          </figure>
        </div>
      </div>
      <div class="item item-wide">
        <div class="item-wrapper">
          <figure class="he-1 amber darken-3" data-link="project-link-2.html">
            <img src="project-2-img.jpg" alt="">
            <figcaption>
              <h4>Project 2</h4>
              <p>tag 1, tag 2, tag 3</p>
            </figcaption>
          </figure>
        </div>
      </div>
    </div>  
  </div>
</section>

For large images, add item-wide class to your item. For tall image, add item-tall to your image.

To change columns number, replace grid-4 by grid-2 for 2 columns, grid-3 for 3 columns...

Grid Layout

Live example

Grid layout as same markup as masonry layout. Just use image with same dimensions and not use item-wide or item-tall classes that are classes dedicated at masonry layout.
Here is an example with 4 columns and caption visible:

<section class="section-portfolio">
  <div class="container-fluid">
    <ul class="items-filter p-l-10">
      <!-- ITEMS FILTERS CONTENT -->
    </ul>
    <div class="masonry grid grid-4 with-caption">
      <div class="item">
        <div class="item-wrapper">
          <figure class="he-2 caption-visible caption-center blue darken-3">
            <img src="project-1-img.jpg" alt="">
            <div class="hover-icons">
              <div class="hover-icons-wrapper">
                <a href="project-1-img.jpg" class="magnific">
                  <i class="nc-icon-outline arrows-1_window-zoom-in"></i>
                </a>
              </div>
            </div>
            <figcaption>
              <h4><a href="project-1.html">Project 1</a></h4>
              <p>tag 1, tag 2, tag 3</p>
            </figcaption>
          </figure>
        </div>
      </div>
      <div class="item">
        <div class="item-wrapper">
          <figure class="he-2 caption-visible caption-center red darken-3">
            <img src="project-2-img.jpg" alt="">
            <div class="hover-icons">
              <div class="hover-icons-wrapper">
                <a href="project-2-img.jpg" class="magnific">
                  <i class="nc-icon-outline arrows-1_window-zoom-in"></i>
                </a>
              </div>
            </div>
            <figcaption>
              <h4><a href="project-2.html">Project 2</a></h4>
              <p>tag 1, tag 2, tag 3</p>
            </figcaption>
          </figure>
        </div>
      </div>
    </div>
  </div>
</section>

To change columns number, replace grid-4 by grid-2 for 2 columns, grid-3 for 3 columns...
