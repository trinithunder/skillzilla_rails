Lateral Navigation

You can define your navigation to the left or right side of the page.

Lateral navigation can have a light, dark or image background.

You can add footer to lateral navigation, with social links or ecommerce features (cart, wishlist, search).

Live example

Basic markup

Here is the basic markup of lateral navigation (left side with light background).

<!-- BEGIN LATERAL NAVIGATION -->
<header id="main-navigation">
  <div class="main-nav-wrapper">
    <div id="logo">
        <a href="#">
          <img src="assets/img/logo/your-logo.png" alt="logo">
        </a>
    </div>
    <nav id="main-menu">
      <!-- YOUR MENU -->
    </nav>
    <footer>
      <a href="" class="facebook">
      <span><i class="fa fa-facebook"></i></span>
      </a>
      <a href="" class="instagram">
      <span><i class="fa fa-instagram"></i></span>
      </a>
      <a href="" class="twitter">
      <span><i class="fa fa-twitter"></i></span>
      </a>
    </footer>
  </div>
</header>
<!-- END LATERAL NAVIGATION -->

Dark background color

To have light navigation, add header-dark class to your body:

<body class="header-transparent">

Image background

If you want to have a background image, add img-cover class to your header id and a data-bg-img:

<div id="header" class="img-cover" data-bg-img="your-image-path.jpg">
  <!-- BEGIN LATERAL NAVIGATION -->
  <header id="main-navigation">
    <!-- MAIN NAVIGATION CONTENT -->
  </header>
  <!-- END LATERAL NAVIGATION -->
</div>

Ecommerce footer

If you want to replace social links by ecommerce links, here is the footer markup:

<footer>
  <a href="#" class="tools-btn" data-toggle-search="fullscreen">
    <span class="tools-btn-icon"><i class="nc-icon-outline ui-1_zoom"></i></span>
  </a>
  <a href="wishlist" class="tools-btn">
    <span class="tools-btn-icon"><i class="nc-icon-outline ui-2_favourite-28"></i></span>
    <span class="wishlist_items_number">0</span>
  </a>
  <a href="cart" class="tools-btn">
    <span class="tools-btn-icon"><i class="nc-icon-glyph shopping_bag-16"></i></span>
    <span class="cart_items_number">0</span>
  </a>
</footer>

Right Navigation

To place navigation on the right side, just add right-nav to your body:

<body class="right-nav">
