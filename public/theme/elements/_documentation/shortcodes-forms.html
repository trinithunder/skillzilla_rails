Form elements

Live example

This template use the default bootstrap form markup. You can read more about it here: http://getbootstrap.com/css/#forms.

Standard Form Layout

To create a basic form, you just have to add form-control class to each input, and wrap label & input inside a form-group class div.

<form role="form">
  <div class="form-group">
    <label for="email">Email address</label>
    <input type="email" class="form-control" id="email">
  </div>
  <div class="form-group">
    <label for="password">Password</label>
    <input type="password" class="form-control" id="password">
  </div>
  <button type="submit" class="btn btn-default">Submit</button>
</form>

Horizontal Form Layout

If you prefer to have label & input on the same line, just add form-horizontal class to your form.

<form role="form" class="form-horizontal">
  <!-- YOUR FORM CONTENT -->
</form>

Normal & grouped style

For each form element, you can choose between classic design or grouped design. To create grouped design, add form-grouped to your form-group class.

<div class="form-group form-grouped">
  <label>Input name</label>
  <input type="text" class="form-control" placeholder="default input">
</div>

Input Size

You can change input size by adding input-sm class for small input or input-lg for large input. You can choose various sizes: small, medium and large input.

<div class="form-group">
  <label>Large</label>
  <input type="text" class="form-control input-lg" placeholder="input-lg">
</div>
<div class="form-group">
  <label>Normal</label>
  <input type="text" class="form-control" placeholder="default input">
</div>
<div class="form-group">
  <label>Small</label>
  <input type="text" class="form-control input-sm" placeholder="input-sm">
</div>

Append & prepend input

Add icons inside your input to explain which type of data is required. You can add all icons available, at the beginning or at the end of input.

<div class="prepend-icon">
  <input type="text" class="form-control" placeholder="Firstname">
  <i class="fa fa-user"></i>
</div>

<div class="append-icon">
  <input type="text" class="form-control" placeholder="Lastname">
  <i class="fa fa-user"></i>
</div>

Select input

Theme use Select2 plugin. It offer a wide range of options and functionnalities.

Select2 plugin website

Basic markup

<select class="form-control">
  <option value="value-1">Value 1</option>
  <option value="value-2">Value 2</option>
  <option value="value-3">Value 3</option>
</select>

Select with search option

<select class="form-control" data-search="true">
  <option value="value-1">Value 1</option>
  <option value="value-2">Value 2</option>
  <option value="value-3">Value 3</option>
</select>

Select with multiple select. Here each selection appear like a tag.

<select multiple class="form-control" data-placeholder="Choose one or various options...">
  <option value="value-1">Value 1</option>
  <option value="value-2">Value 2</option>
  <option value="value-3">Value 3</option>
</select>

Grouped layout.

<div class="form-group form-grouped form-grouped-select-2">
  <label>Multiple Select</label>
  <select multiple class="form-control">
    <!-- OPTIONS HERE -->
  </select>
</div>

Tags input

Tags input are usefull if you want to allow visitor to enter various custom values. You can add default value with value attribute

<div class="form-group">
  <input class="select-tags form-control" value="brown,red,green">
</div>

Grouped layout.

<div class="form-group form-grouped">
  <label>Select favorite colors</label>
  <input class="select-tags form-control" value="brown,red,green">
</div>

Datepicker

Theme use Bootstrap Datepicker plugin. It's one of the most used datepicker plugin.

Bootstrap Datepicker page

Here is an example with a prepend icon (optional):

<div class="prepend-icon">
  <input type="text" class="form-control datepicker" placeholder="choose a date">
  <i class="fa fa-calendar"></i>
</div>

Grouped layout.

<div class="form-group form-grouped">
  <label>Select a Date</label>
  <div class="prepend-icon">
    <input type="text" class="form-control datepicker" placeholder="choose a date">
    <i class="fa fa-calendar"></i>
  </div>
</div>

Radio input

There is 2 layouts: horizontal & vertical.

Horizontal layout (list):

<div class="radio">
  <input type="radio" name="radio-button-1" id="radio-1" checked>
  <label for="radio-1">Choice 1</label>
</div>
<div class="radio">
  <input type="radio" name="radio-button-1" id="radio-2">
  <label for="radio-2">Choice 2</label>
</div>
<div class="radio">
  <input type="radio" name="radio-button-1" id="radio-3">
  <label for="radio-3">Choice 3</label>
</div>

Vertical layout:

<ul class="form-list">
  <li>
    <input type="radio" name="radio-button-3" id="radio-7" checked>
    <label for="radio-7">Choice 1</label>
  </li>
    
  <li>
    <input type="radio" name="radio-button-3" id="radio-8">
    <label for="radio-8">Choice 2</label>
  </li>

  <li>
    <input type="radio" name="radio-button-3" id="radio-9">
    <label for="radio-9">Choice 3</label>
  </li>
</ul>
