Footer

Footer comes with many styles variation and a large set of widgets.

Boxed or Wide

Widgets inside footer can be boxed or wide. If you choose boxed version, footer content will have a maximum width of 1200px.

For boxed version, add container class to footer-main and footer-bottom elements.

<footer id="footer">
  <div id="footer-main" class="container">
    // FOOTER WIDGETS
  </div>
  <div id="footer-bottom">
    <div class="container">
        // FOOTER COPYRIGHT
    </div>
  </div>
</footer>

If you prefer a wide layout, replace container class by container-large.

Footer widgets

Theme offer a very large set of possibilities. Here is the basic markup of a widget:

<div class="widget">
  <h3 class="widget-title">Widget title</h3>
  <!-- WIDGET CONTENT -->
</div>

Here is the list of all widgets available for footer.

Text widget

Here you can add text paragraph as well as images and links.

Twitter widget

This widget will display latest tweets from account you choose with animation.

Menu Widget

Use this one if you want to show some important links, like categories, terms...

Flickr widget

This widget will display pics from Flickr account you choose with animation.

Dribbble widget

This widget will display pics from Dribbble account you choose with animation.

Posts widget

Display your latest posts here. Can be useful for blog and corporate website.

Products widget

Display your latest or best products here. Specific for ecommerce website.

Newsletter widget

Add a form in your footer to encourage user to subscribe to your newsletter.

Advertisement widget

You can add a banner too in order to get some money from ads.

Facebook widget

Add a Facebook widget to encourage user to like your page.

Google Plus widget

Add a Google Plus widget to encourage user to follow your page.

Contact widget

Add a form in your footer to encourage user to get in touch with you.

Map widget

Add map to show where is your office for example.

Video widget

If you want, it's possible to add a embed video (Youtube or Vimeo) in footer too.

Copyright

You can add copyright information inside footer-bottom element. In this part, you can add some quick links too, as well as small images like payment logos for example.
