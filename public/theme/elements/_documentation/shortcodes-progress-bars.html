Progress bars

Animated progress effect for skills, works or project for example.

Live example

Basic markup

<div class="progress">
  <div class="progress-bar" role="progressbar" value="91" aria-valuemin="0" max="100">   
    <div class="progress-info">
      <span class="progress-name">Design</span>
      <span class="progress-value">91%</span>
    </div>
  </div>
</div>

Progress bar have 3 main attributes:

aria-valuemin: define minimum value of progress bar (0 in general)

aria-valuemax: define maximum value of progress bar (often 100)

aria-valuenow: current value of this skills, works...

Animation delay

To add animation delay, add data-animation-delay to progress bar element. For example, for a 1.5 second delay:

<div class="progress">
  <div class="progress-bar" role="progressbar" value="91" aria-valuemin="0" max="100" data-animation-delay="1500">   
    <div class="progress-info">
      <span class="progress-name">Design</span>
      <span class="progress-value">91%</span>
    </div>
  </div>
</div>
