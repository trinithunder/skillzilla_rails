Theme Colors

It's very important that you can easily customize theme to your needs. For that reason, we include various colors theme.

List of themes color

Just add this class to your body to apply theme. Here is the list of current list of theme color included:

  • Blue theme: theme-blue
  • Amber theme: theme-amber
  • Indigo theme: theme-indigo
  • Purple theme: theme-purple
  • Teal theme: themes/theme-teal
  • Red theme: theme-red
  • Brown theme: theme-brown
  • Dark theme: theme-dark

How to use a theme color?

To use a theme color, add theme-blue class to your body for example.

<body class="theme-blue">
