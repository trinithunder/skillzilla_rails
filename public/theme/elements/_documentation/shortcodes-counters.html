Counters

Animated up and down effect.

Live example

Count Up

Theme use CountTo plugin. For more information about it, please visit plugin official website.

Count Up markup

<div class="counter">
  <i class="nc-icon-outline food_coffee"></i>
  <div class="countup" data-from="0" data-to="1354" data-speed="1000">0</div>
  <p class="count-title">Cups of coffee</p>
  <p class="count-subtitle">To think fast and fine</p>
</div>

Icon, title and subtitle are optional.

Countup have 3 parameters:

data-from: begin number

data-to: end number

data-spped: speed in millisecond

Count Up size

For bigger count up style, add counter-lg class to counter element:

<div class="counter counter-lg">
  <i class="nc-icon-outline emoticons_quite-happy"></i>
  <div class="countup" data-from="0" data-to="1354" data-speed="1000">0</div>
  <p class="count-title">Satisfied customers</p>
  <p class="count-subtitle">Carefully craft design</p>
</div>

Countdown

Theme use Countdown plugin. For more information about it, please visit plugin official website.

Countdown markup

<div class="countdown" data-date="2020/10/10"></div>

Countdown have only 1 parameter: date. Change value to date you want.

Inline countdown

For inline countdown, add countdown-inline class to countdown element. Example:

<div>This offer will end in: <div class="countdown countdown-inline" data-date="2016/10/10"></div></div>
